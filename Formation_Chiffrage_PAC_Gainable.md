## PHASE 2 : DIMENSIONNEMENT

### 2.1 Calcul des besoins thermiques

**Formule simplifiée :**
```
Puissance (kW) = Surface (m²) × Coefficient de déperdition
```

**Coefficients selon le type de bâtiment :**
- Ancien bâtiment mal isolé : 0,15-0,20 kW/m²
- Bâtiment standard : 0,10-0,15 kW/m²
- Bâtiment bien isolé (RT 2012+) : 0,05-0,10 kW/m²

### 2.2 Calcul par exemple

```
Surface totale : 70 m²
Coefficient : 0,12 kW/m²
Puissance nécessaire = 70 × 0,12 = 8,4 kW
→ Sélectionner une PAC de 9 kW (arrondir à l'unité supérieure)
```

## PHASE 3 : SÉLECTION DES ÉQUIPEMENTS

### 3.1 Unité Extérieure

**Caractéristiques à vérifier :**
- Puissance frigorifique (kW)
- Puissance calorifique (kW)
- Débit d'air nominal (m³/h)
- Bruit (dB)
- Dimensions et poids
- Type de fluide frigorigène
- Classe énergétique (A+++, A++, A+)

**Checklist d'installation :**
- [ ] Encastrement possible dans une niches
- [ ] Installation en toiture ou au sol
- [ ] Dégagement minimum 1 m sur tous les côtés
- [ ] Accès pour entretien

### 3.2 Unité Interne (Module Gainable)

**Critères de sélection :**
- Puissance compatible avec l'unité extérieure
- Débit d'air (m³/h) adapté aux conduits
- Dimensions (longueur × hauteur × profondeur)
- Niveau sonore (db(A))
- Type de régulation compatible

**Modèles courants :**
- Gainable standard horizontal : 800-1200 m³/h
- Gainable compact : 600-900 m³/h
- Gainable haute performance : 1200-1500 m³/h

### 3.3 Système de Régulation

#### 3.3.1 Régulation simple (2 zones)

- **Thermostat filaire mural** : Contrôle on/off ou modulation
- **Clapet de zone manuel** : Séparation zone 1 / zone 2
- **Coût** : 300-600 €

#### 3.3.2 Régulation multi-zone avec clapets motorisés

- **Thermostat mural programmable** : Affichage numérique, programmation
- **Clapets motorisés** (1 par zone) : Fermeture/ouverture automatique
- **Capteurs de température** : Un par zone
- **Centrale de régulation** : Gestion des clapets
- **Coût par zone supplémentaire** : 150-250 €

#### 3.3.3 Régulation zone par zone (Luxe)

- **Module de gestion intelligente** : Application mobile, géolocalisation
- **Clapets motorisés équipés de capteurs** 
- **Humidificateur/déshumidificateur** optionnel
- **Coût de base** : 1000-1500 € + 100-150 € par zone

**Choix recommandé selon configuration :**

| Type de bâtiment | Recommandation | Budget régulation |
|---|---|---|
| T2-T3 (< 50m²) | Régulation simple | 400-600 € |
| T4-T5 (50-100m²) | 2-3 zones motorisées | 800-1200 € |
| Maison + étages | Multi-zone intelligente | 1500-2500 € |

### 7.2 Calcul des marges

**Formule générale :**
```
Coût de revient = Coût matériel + Coût MOD + Frais généraux
Marge brute = Prix de vente - Coût de revient
Taux de marge = (Marge brute / Coût de revient) × 100
Coefficient multiplicateur = Prix de vente / Coût de revient
```

**Exemple chiffré :**
```
Matériel .............................. 4000 €
Main-d'œuvre .......................... 1000 €
Frais généraux (10%) .................. 500 €
COÛT DE REVIENT = 5500 €

Taux de marge visé : 40%
Prix de vente = 5500 € × (1 + 0,40) = 7700 € HT
                                       = 9240 € TTC

Vérification :
Marge brute = 7700 € - 5500 € = 2200 €
Taux = 2200 / 5500 = 40% ✓
```
